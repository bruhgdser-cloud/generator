# Nick-Gen 🎲

Легкий и быстрый генератор никнеймов на Lua (LuaJIT) для Arch Linux.

## Установка

Клонируйте репозиторий и установите пакет:

```bash
git clone https://github.com/bruhgdser-cloud/generator.git
cd generator
sudo pacman -U nick-gen-1.0.0-1-any.pkg.tar.zst
```

Или соберите из исходников:

```bash
git clone https://github.com/bruhgdser-cloud/generator.git
cd generator
makepkg -si
```

## Использование

```bash
nick-gen
```

## Зависимости
- `luajit`
